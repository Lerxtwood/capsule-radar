Capsule Radar + TamaPoke

Combined firmware for the Waveshare ESP32-S3 Touch AMOLED 1.75: live ADS-B radar plus the TamaPoke pet app, switchable on-device.

1. Install firmware

Uses the latest GitHub Release full-flash image

Flash from Chrome or Edge - no PlatformIO or Arduino install needed.

Use Chrome or Edge on desktop. Web Serial is required. This page must be served over HTTPS or localhost.

First-time installs should use Erase device + Install. Later reinstalls can skip erase if you want to keep saved settings and TamaPoke progress.

2. Configure WiFi

  1. Plug the board in with a USB-C data cable.
  2. Click Install combined firmware and choose the USB JTAG/serial port.
  3. After reboot, join the CapsuleRadar-Setup WiFi network from your phone or computer.
  4. Enter your WiFi and radar home location.
  5. Fine-tune settings later at http://capsuleradar.local/.

3. TamaPoke sprites

The combined firmware already supports TamaPoke sprites from the microSD card. The browser sprite loader is the next piece to wire into this page.

  • Firmware-side serial SD loading support is present.
  • The next iteration will fetch sprite assets and copy them to /mons on the SD card.

For now, install the firmware here, then we'll add the one-click sprite copy flow after the first flasher test.
